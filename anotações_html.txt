### Significado do HTML
    -> HTML (HyperText Markup Language) é uma linguagem de marcação de hipertexto usada para criar e estruturar o conteúdo de páginas da web. Ele define elementos que representam diferentes tipos de conteúdo, como textos, imagens, links e multimídia, e fornece a estrutura e a semântica necessárias para a apresentação e interpretação do conteúdo em navegadores da web. O HTML é a base da maioria das páginas da web e permite aos desenvolvedores criar páginas interativas e acessíveis.



### Modelo de Caixa (Cada elemento visível em um site é encapsulado em uma caixa)
    ## height (Altura)                  -> Define a dimensão vertical da caixa.
    ## width (Largura)                  -> Define a dimensão horizontal da caixa.

    ## padding (Preenchimento)          -> Espaço entre a caixa (box), do elemento e a borda do elemento
        # padding-top / padding-left / padding-bottom / padding-right       -> Define o preenchimento de cada lado do padding
        - Shorthand (Abreviação de atributos)
            padding: <padding-top> <padding-left> <padding-bottom> <padding-right>          Exp: padding: 10px 10px 10px 10px; | OU | padding: 10px;

    ## border (Borda)                   -> Cria uma borda ao redor do padding.
    ## outline (Contorno)               -> Cria um contorno ao redor da borda que consome espaço da margem.

    ## margin (Margem)                  -> Define a margem que separa o elemento de outros elementos.
        # margin: auto;         -> Centraliza o elemento no meio da box.
        # margin-top / margin-left / margin-bottom / margin-right       -> Define o preenchimento de cada lado do margin
        - Shorthand (Abreviação de atributos)
            margin: <margin-top> <margin-left> <margin-bottom> <margin-right>          Exp: margin: 10px 10px 10px 10px; | OU | margin: 10px;

    ## Tipos de caixa
        # Nível de Bloco (box-level): Elementos que ocupam a largura total da tela. Exemplos incluem:
            - <div>             -> Usado para agrupar conteúdo em um bloco.
            - <h1>-<h6>, <p>, <main>, <header>, <nav>, <article>, <aside>, <footer>, <form>, <video>    -> Elementos que representam seções ou estruturas de nível de bloco em uma página web.

        # Nível Inline (inline-level): Elementos que ocupam apenas a largura especificada. Exemplos incluem:
            - <span>            -> Usado para aplicar estilos a partes específicas ou conteúdo em linha.
            - <a>, <code>, <small>, <strong>, <em>, <sup>-<sub>, <label>, <button>, <input>, <select>   -> Elementos que representam conteúdo em linha, como links, texto em destaque e campos de entrada.



### Tags
    ## Tags PADRÃO
        <!DOCTYPE html>         -> Indica a versão do HTML usada no código (por padrão, HTML5).
        <html>                  -> Define a área de conteúdo HTML
        <!-- --!>               -> Define uma área para fazer comentários sobre o código

    ## Tags HEAD (Define a seção de metadados do documento, incluindo informações sobre a página, tais como título, codificação de caracteres, e conexões com arquivos CSS ou ícones.)
        <meta charset="UTF-8">                                                      -> Define a codificação de caracteres a ser utilizada pelo site, permitindo a exibição correta de caracteres e acentuação.
        <meta name="viewport" content="...">                                        -> Informa ao navegador que o conteúdo do <body> utilizará 100% do espaço disponível.
        <meta name="description">                                                   -> Define uma descrição da página para os mecanismos de busca
        <title>                                                                     -> Define o título da página exibido na barra de título do navegador.
        <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">           -> Define o favicom da página
        <link rel="stylesheet" href="style.css">                                    -> Define a folha de estilização da página (CSS)
        <style>                                                                     -> Define uma área para adicionar estilos CSS diretamente no arquivo HTML.

    ## Tags semânticas  (São semelhantes a divs, porém tem utilizações especificas para melhor organização semântica)
        <header>            -> Define o conteúdo do cabeçalho do site
        <main>              -> Define o conteúdo principal
        <footer>            -> Define o rodapé do site
        <section>           -> Define uma seção para o site
        <article>           -> Define um conteúdo para o site
        <aside>             -> Conteúdo periférico ao artigo

    ## Tags BODY (Definea a área de conteúdo visível em uma página da web. Incluindo texto, imagens, links e outros elementos visuais)
        # Elementos de Texto:
            <h1> - <h6>         -> Títulos de diferentes níveis.
            <p>                 -> Parágrafos de texto.

        # Elementos de formatação de texto:
            <em>                -> Usado para destacar texto com ênfase.
            <address>           -> Informações de endereço.
            <strong>            -> Usado para denotar um texto de grande importância
            <cite>              -> Indica que o texto é uma referência ou citação de outra fonte
            <abbr>              -> Define uma abreviação e fornece um texto explicativo quando o usuário passa o mouse sobre
            <time>              -> Usado para representar datas e horas em um formato específico
            <code>              -> Define texto com fonte monoespaçada.
            <pre>               -> Pré-formatação de texto.

        # Elementos de Mídia:
            <img>                   -> Insere imagens.
            <audio>                 -> Reproduzir áudio com opções como preload, muted, autoplay, controls e loop.

        # Outros Elementos:
            <br>                -> Quebra de linha.
            <a href="">         -> Cria links âncora.



### Atributos comuns
    ## ID's e classes
        # ID (Nome de identificação único definido a um único elemento)
            - <div id="div-principal">          -> No CSS acessamos o elemento através de #         -> Exemplo de CSS: div#div-principal {}

        # Classe (Nome de identificação de uma classe a ser personalizada, mais de um elemento pode receber a mesma classe)
            - <h1 class="texto-principal">          -> No CSS acessamos o elemento através de .         -> Exemplo de CSS: .texto-principal {}
            - <h2 class="texto-principal">

    ## Conexões entre arquivos
        # src           -> O atributo src é comumente usado em elementos de mídia, como <img> e <audio>, para especificar o caminho do arquivo de origem.           -> Exp: <img src="imagem.jpg">
        # href          -> O atributo href é usado em elementos de âncora (<a>) para definir o URL de destino do link.                                              -> Exp: <a href="https://www.exemplo.com">Aqui</a>

    ## Acessibilidade
        # alt           -> O atributo alt é usado em elementos de imagem (<img>) para fornecer uma descrição alternativa da imagem.         -> Exp: <img src="foto.jpg" alt="Família se divertindo em um parque de diversões">



### Multimídia
    ## Dicas
        - Atributo alt para Imagens: Todos os elementos de imagem <img> devem incluir o atributo alt, que fornece uma descrição alternativa para a imagem. Isso é essencial para acessibilidade e ajuda os usuários com deficiência visual a compreender o conteúdo da imagem.
        - Tamanho de Mídias: O tamanho das mídias, como imagens, vídeos e áudios, é crucial. Imagens pesadas podem resultar em tempos de carregamento mais longos, afetando a experiência do usuário. Sempre otimize as imagens para a web.
        - Midias Locais vs. Remotas: Midias locais precisam ser baixadas pelos clientes e consomem largura de banda do servidor. Midias remotas não consomem largura de banda do servidor, mas a disponibilidade pode não ser garantida.
        - Formato de Vídeo: Para vídeos locais, use formatos como MP4, WEBM e MKV para garantir a compatibilidade com diversos navegadores. Evite arquivos WAV devido ao seu tamanho.
        - Vídeos Remotos: Ao usar vídeos remotos, considere serviços como o YouTube (gratuito, mais rápido, mas menor qualidade) ou o Vimeo (pago, maior qualidade e personalização).    

    ## Imagens
        # Imagens Locais (Imagens locais precisam ser baixadas pelos clientes e consomem banda do servidor)
            - Use formatos como PNG e JPG.

        # Imagens Remotas (Imagens remotas não consomem banda do servidor porém não podemos garantir a disponibilidade da imagem)
            - Referencie imagens hospedadas em servidores externos.

    ## Vídeos
        # Vídeos Locais (Vídeos locais precisam ser baixadas pelos clientes e consomem banda do servidor)
            - Aceitam formatos como MP4, WEBM, MKV e WAV.

        # Vídeos Remotos (Vídeos remotos não consomem banda do servidor porém não podemos garantir a disponibilidade do vídeo)
            - Utilize serviços como YouTube (gratuito, menos qualidade e mais rápido) ou Vimeo (pago, mais qualidade e maior customização).

    ## Áudios
        Formatos aceitos incluem MP3, OGG e WAV.



### Listas (Itens para listas ordenadas e não ordenadas são definidos por <li> </li>)
    ## Listas Ordenadas (ordem dos elementos importa)
        <ol type="">            -> Cria listas ordenadas com tipos como 1, A, a, I, i.

    ## Listas Não Ordenadas (ordem dos elementos não importa)
        <ul type="">            -> Cria listas não ordenadas com tipos como disc, circle e square.

    ## Listas de Definição (Listas para atribuir uma coisa a outra)
        <dl>            -> Lista de definição.
        <dt>            -> Termo ou título da definição.
        <dd>            -> Definição do termo.
